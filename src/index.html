Cycler

Cycler is a high-throughput cell cycle analysis program used for analyzing results from flow cytometry or fluorescent microscopy automatically with limited human input for maximum reproducibility.

Begin by selecting .FCS or .CSV files containing fluorescent DNA data.

About Cycler

Cycler was designed for the purpose of making cell-cycle analysis fast and efficient across many experiments conducted at the same time. As such, it relies on consistent data.

Feel free to drop by our Github Repository to report any issues or view the source code.
